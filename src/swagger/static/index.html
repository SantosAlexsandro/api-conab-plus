G4Flex - Integração

Autenticação

Para utilizar a API G4Flex, é necessário obter um token JWT através do endpoint de autenticação:

POST /api/integrations/g4flex/token

Fluxo de Autenticação:

  1. Faça uma requisição POST para /api/integrations/g4flex/token fornecendo apiKey e clientId válidos
  2. O sistema retornará um token JWT
  3. Inclua este token no header Authorization de todas as requisições subsequentes: Bearer {seu-token}
Nota: O token expira conforme configurado no ambiente. Verifique a documentação da API para detalhes sobre o tempo de expiração.

Endpoints Disponíveis

  • Contratos: Verificação de contratos ativos.
  • Ordens de Serviço: Verificação, solicitação e fechamento de ordens de serviço.
